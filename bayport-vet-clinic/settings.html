Security

Device ID:

    Email (MFA & reminders)

    Online (Render): Render FREE blocks Gmail SMTP. Use Brevo (recommended) or Resend with a verified domain.

    1. Sign up at brevo.com (free — 300 emails/day).
    2. Senders → Add sender — verify bayport.vetclinic@gmail.com (check inbox for Brevo link).
    3. SMTP & API → API Keys — create key, copy it.
    4. In Render → bayport-api → Environment, set:
      BREVO_API_KEY = your key
      BREVO_FROM_EMAIL = verified sender (e.g. bayport.vetclinic@gmail.com)
      Redeploy Render.

    Resend only: onboarding@resend.dev can email your Resend login only. To use Resend for pet owners, verify a domain at resend.com/domains and set RESEND_FROM.

    Local desktop: Gmail app password in data/mail.env still works.

    If email is not configured, MFA codes appear in Admin notifications (bell icon) instead.

    Inbox profile logo (circle next to sender name, like Paw Care) is set on your Google account profile photo when sending from Gmail, or via BIMI + your own domain on Resend — not inside the email template. See docs/EMAIL_SENDER_AVATAR.md in the repo.
